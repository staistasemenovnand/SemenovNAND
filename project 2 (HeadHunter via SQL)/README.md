### Описание проекта    
Компания HeadHunter предоставила базу вакансий. Требуется проанализировать
структуру данных, вывести основные описательные статистики, оценить наличие
зависимостей между параметрами вакансии. 

:arrow_up:[к оглавлению](_)


### Какой кейс решаем?    

Применяем инструменты работы с базами данных через SQL и применением
соединения таблиц, расчета агрегатных показатей, сортировки, ограничения
вывода и объединения нескольких выборок. Дополнительно используем методы
работа с датафреймами и визуализации данных.


### Краткая информация о данных
В качестве источника данных используется база данных project_sql на портале METABASE.
